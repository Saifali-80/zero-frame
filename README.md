# ZERO FRAME

**Typography Before Interface. Content Beyond Containment.**

A Swiss International Style portfolio template for creative agencies and design studios. Zero images. Zero CSS frameworks. Pure typography and grid-based design.

## ✦ Philosophy

- **Content over decoration** — meaning before design
- **Typography as interface** — no images, no distractions
- **Swiss grid system** — asymmetric layouts, clear hierarchy
- **Single file deployment** — everything inline, zero dependencies

---

## ✦ Features

- ⚡ **Single HTML file** — no build tools, no npm, no frameworks
- 🎨 **Swiss International Style** — clean typography, asymmetric grids
- ♿ **Accessible** — skip links, focus trap, keyboard navigation
- 📱 **Fully responsive** — mobile-first with sidebar + drawer navigation
- ✨ **Reveal animations** — Intersection Observer based
- 🔄 **Marquee ticker** — infinite scroll animation
- 🗂️ **Accordion services** — smooth expand/collapse
- 📊 **Scroll progress bar** — reading progress indicator
- ⬆️ **Back to top** — smooth scroll button
- 📋 **Copy email** — one-click clipboard
- 📬 **Contact form** — Formspree integration ready
- 🎵 **Minimal sound design** — subtle hover/click feedback
- ⌨️ **Noise toggle** — press `G` to toggle grain texture
- 🔢 **Count-up animation** — scroll-triggered number animation
- 👻 **Ghost numerals parallax** — subtle scroll effect
- 📍 **Floating mobile CTA** — always-visible contact button
